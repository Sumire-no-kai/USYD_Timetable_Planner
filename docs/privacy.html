Privacy Policy

Effective date: 6 July 2026

USYD Timetable Planner is a browser extension that helps users capture timetable options from Sydney Timetable, compare class times, avoid clashes, and plan preference order locally in the browser.

Independence

USYD Timetable Planner is an independent student-made tool. It is not affiliated with, endorsed by, or officially supported by the University of Sydney.

Data Handled by the Extension

The extension reads timetable table content from Sydney Timetable pages when the user clicks the capture button. This may include course codes, activity types, class times, locations, availability, teaching weeks, and related timetable information shown on the page.

How Data Is Stored

Captured timetable data, language preference, and planner settings are stored locally in the user's browser using chrome.storage.local.

How Data Is Used

The data is used only to generate timetable planning suggestions, clash checks, preference order suggestions, weekly previews, and CSV exports.

Data Sharing

The extension does not sell, transfer, or share user data with third parties. It does not use a backend server, analytics service, advertising service, or tracking service.

Remote Code

The extension does not use remote code. All extension JavaScript is included in the installed extension package.

External Links

The extension may include optional feedback or support links. These links open external websites only when clicked by the user. Those external websites are governed by their own privacy policies.

User Control

Users can clear captured timetable data in the extension by using the clear data control. Users can also remove extension data through their browser's extension or site storage settings.

Contact

For questions, feedback, or support requests, please use the support page linked from this site.
