USYD Timetable Planner

Capture Sydney Timetable options, compare class times, avoid clashes, and export a planning reference from your own browser.

Read privacy policy Get support
Not an official University of Sydney service. This is an independent student-made tool. Always confirm final preferences in Sydney Timetable before submitting choices.
Runs locally
Captured timetable data and planner settings stay in the user's browser storage.
No tracking
No backend server, account system, analytics service, advertising service, or tracking service.
Planning only
The extension helps prepare a timetable plan. It does not submit, modify, or guarantee official allocations.

What it reads

The extension reads class option tables from Sydney Timetable only after the user clicks the capture button. The captured table content may include unit codes, activity types, class times, locations, availability, and teaching weeks.
