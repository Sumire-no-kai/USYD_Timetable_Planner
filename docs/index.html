USYD Timetable Planner

A local-first browser extension for planning University of Sydney timetable preferences.

Independent project. This extension is not affiliated with, endorsed by, or officially supported by the University of Sydney.

What it does

USYD Timetable Planner reads class option tables from Sydney Timetable after the user clicks the capture button. It helps compare class times, avoid timetable clashes, preview a weekly timetable, and export a readable CSV plan.

Important limits

  • It does not submit or change official timetable preferences.
  • Generated plans are for planning reference only.
  • Users should manually confirm times, locations, availability, teaching weeks, and final preference order in Sydney Timetable.

Privacy summary

  • No backend server.
  • No account system.
  • No analytics, advertising, or tracking.
  • Captured timetable data is stored locally in the user's browser.
